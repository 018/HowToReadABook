# 如何阅读一本书
HowToReadABook是Zotero的一个插件，它根据《如何阅读一本书》的四个阅读层级及笔记法而来。如需自定义模版，可参考另一个插件[zotcard](https://github.com/018/zotcard)。

[下载插件](https://github.com/018/HowToReadABook/releases)
